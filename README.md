# Beach Houses In Toco

## Purpose

This is the repository for an online booking agency that specifically deals with bookings in Toco Trinidad. The webpage consists of the following on a SPA:

- A Homepage
- An About Section
- A Contact Page

There also exists a separate blog page.

## Instructions

A user can make a booking by clicking the "Book Now" CTA button on the homepage or by scrolling down directly to the booking page. In addition, a user can also click on an internal link located in the blog article. After entering the relevant credentials on the booking page (name, email, check-in, check-out and guest) they must then click the check for available booking CTA which submits the data to backend and customer is then subsequently emailed with a payment link and images of the room to be booked.
